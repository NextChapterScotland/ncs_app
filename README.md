# SH26 Main

## Name
Next Chapter Scotland App

## Description
The project is a mobile app for Next Chapter Scotland which contains an Information Hub section containing resources from Next Chapter Scotland's website and a forum where users can either browse as guests or log in and contribute to the discussions themselves.

## How to Run - WINDOWS

1. Install an IDE (VSCode recommended - https://code.visualstudio.com/download)

2. Install Flutter at https://docs.flutter.dev/install . There are guides to help you through the installation of flutter, we recommend clicking on the 'Quick Start' option.

3. Install Git at https://git-scm.com/install/ . This will allow you to clone the repository.

4. Install Android Studio at https://developer.android.com/studio

5. Install Visual Studio at https://visualstudio.microsoft.com/vs/

6. Clone the repository on Git with git clone [repository URL]. Make sure you are in a folder that you will remember, e.g.  C:Users/your-username.

7. Go to the folder you cloned the repository in with cd [folder URL] (e.g. cd C:Users/your-username) and run `flutter pub get`

8. Check if there any any issues by running `flutter doctor`. You may need to update some things or research some issues.

9. Choose an emulator at the bottom right where it says `No Device` - at the top click `Start Medium Phone`.

10. Go onto main.dart and click the play button in the top right corner.

## How to Run - MAC

1. Install an IDE (Visual Studio Code recommended - https://code.visualstudio.com/download)

2. Install Flutter at https://docs.flutter.dev/install There are guides to help you through the installation of Flutter — we recommend clicking on the “Quick Start” option for macOS.

3. Install Git at https://git-scm.com/download/mac This will allow you to clone the repository.

4. Install Android Studio at https://developer.android.com/studio (This is needed for Android emulators and SDK tools.)

5. (Optional but recommended) Install Xcode from the Mac App Store (This is required if you want to run iOS simulators.)

6. Clone the repository using Git: `git clone [repository URL]` Make sure you are in a folder you will remember, e.g.: /Users/your-username

7. Go to the folder you cloned the repository in: cd /Users/your-username/[project-folder]. Then run: `flutter pub get`

8. Check if there are any issues by running: `flutter doctor`. You may need to install additional dependencies or follow suggestions shown in the terminal.

9. Choose an emulator: Open Android Studio → Device Manager → Start a virtual device OR Use an iOS simulator via Xcode (e.g. iPhone 15)

10. Open main.dart in your IDE and click the Run / Play button (top right in VS Code).
